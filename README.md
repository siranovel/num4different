## install ##

From rubygems:  
~~~
    [sudo] gem install num4diff
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4different.git  
    cd num4different  
    gem build num4diff.gemspec
    gem install num4diff
~~~
